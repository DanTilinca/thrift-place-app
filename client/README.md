# Thrift Place (Vite + React)

## Scripts

From the `client/` folder:

### `npm run dev`

Runs the app in dev mode at `http://localhost:3000`.

### `npm run build`

Builds a production bundle to `dist/`.

### `npm run preview`

Previews the production build locally.

### `npm test`

Runs frontend tests with Vitest.
